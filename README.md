# 🚀 Featured Projects

## 📚 Klassbook AI — Multimodal AI Tutor for Textbooks
🔗 https://github.com/inareshmatta/klassbook-ai

Modern studying relies on **static textbooks and PDFs**, forcing students to leave their study context to search for help.

**Klassbook AI transforms textbooks into an interactive AI tutor.**

Instead of switching to another app, the AI:

- Watches the **exact PDF page the student is studying**
- Understands **diagrams and charts**
- Listens to the **student's voice in real-time**
- Responds with **natural voice tutoring**

### Key Capabilities

🎙 **Real-Time Spoken Tutor**  
Powered by **Gemini 2.5 Flash Native Audio API** for low-latency voice interaction.

📄 **PDF Context Awareness**  
The AI understands the exact page and visual content being studied.

🧠 **Multimodal Learning Support**  
Combines **text, diagrams, and speech** for contextual tutoring.

Goal: **Turn static textbooks into AI-guided learning experiences.**

## 💎 iJEMS — Jewellery Enterprise Management System
🔗 https://github.com/inareshmatta/iJems

**iJEMS (Integrity Jewellery Enterprise Management System)** is a full desktop ERP designed specifically for jewellery businesses.

Built with a **local-first architecture**, it ensures that businesses can operate **securely and offline without cloud dependency.**

### Core Modules

🏢 **Store Operations**

- Smart inventory tracking (SKU, purity, weight)
- Barcode-enabled checkout
- Dynamic POS with GST calculation
- Customer CRM with purchase history

💰 **Financial Suite**

- Gold exchange calculations
- Pawn broking workflow
- Sales analytics and stock valuation
- Business dashboards and reports

🤖 **AI & Security**

- OCR scanning for handwritten rate cards
- Role-based access control
- AES-256 encrypted data
- Complete audit trail

### Technology Stack

| Layer | Technology |
|------|-------------|
| Core | Electron |
| Backend | Node.js / Express |
| Database | PostgreSQL |
| AI | Tesseract.js / Local LLM |
